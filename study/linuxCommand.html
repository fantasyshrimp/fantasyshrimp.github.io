리눅스 명령어s


파일 시스템 탐색 기본 명령어

    pwd :
  • 현재 작업중인 디렉토리를 출력
  • $ pwd
  • => /home/vagrant/git/bitcamp-test

    cd :
  • 작업 디렉토리의 위치 변경
  • $ cd /git/bitcamp-study

    ls :
  • 현재 작업중인 디렉토리의 파일 목록 출력
  • $ ls
  • => file1.txt file2.txt g1.txt README.md

    file :
  • 파일의 종류와 속성값 출력
  • $ file a.txt
  • => a.txt: ASCII text

    less :
  • 파일의 종류와 속성값 출력
  • $ less a.txt
  • => 1234 a.txt(END)


파일과 디렉토리 조작 명령어

    cp :
  • 파일 복사
  • $ cp [복사할 파일명] [복사된 파일명]
  • $ cp [복사할 파일명] [복사할 디렉토리]

    mkdir :
  • 디렉토리 생성
  • $ mkdir [디렉토리명]

    rm :
  • 파일 삭제
  • $ rm [파일명]
  • $ rm -rf [디렉토리명]

    ln :
  • 파일의 링크 생성
  • $ ln [파일명] [생성할링크명]
  • => 링크를 통해 파일에 접근가능

명령어를 다루는 명령어

    type :
  • 명령어의 상태 확인
  • $ type test
  • => test is a shell builtin

    which :
  • 명령어의 실행되는 위치 확인
  • $ which ls
  • => /usr/bin/ls

    man :
  • 명령어의 매뉴얼 확인
  • $ man mv

  • MV(1) User Commands MV(1)
    NAME
    mv - move (rename) files

    SYNOPSIS
    mv [OPTION]... [-T] SOURCE DEST
    mv [OPTION]... SOURCE... DIRECTORY
    mv [OPTION]... -t DIRECTORY SOURCE...

    DESCRIPTION
    Rename SOURCE to DEST, or move SOURCE(s) to DIRECTORY.
    Mandatory arguments to long options are mandatory for short options too.
    --backup[=CONTROL]
    make a backup of each existing destination file
    -b like --backup but does not accept an argument ....


    apropos :
  • 명령어 검색
  • $ apropos [검색어]

    info :
  • 명령어의 상세 메뉴얼 확인
  • $ info [명령어]

    whatis :
  • 명령어의 간략한 설명 확인
  • $ whatis ls

  • =>ls (1) - list directory contents
    ls (1p) - list directory contents

      • alias :
    • 명령어의 별칭 등록
    • $ alias mkafile='touch a.txt'
    • => mkafile 입력 시 touch a.txt명령어 실행
